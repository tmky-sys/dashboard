# スマホからClaudeCodeを使うガイド

## ClaudeCodeとは

ClaudeCodeはブラウザから使えるAIコーディングアシスタントです。
スマートフォンからでも、日本語でそのまま話しかけるだけでコードの修正・追加ができます。

---

## スマホからできること

### コードの質問・説明
「このコードは何をしていますか？」と聞くだけで説明してくれます。

### バグ修正
「〇〇が表示されない」「エラーが出る」と報告すると原因を調べて直してくれます。

### 機能追加
「天気予報に週間予報を追加して」など自然な日本語で指示できます。

### ファイルの作成・編集
コードを書かなくても、指示するだけでファイルを作成・編集してくれます。

### Gitへの保存（コミット・プッシュ）
変更内容を自動でGitにコミット＆プッシュするところまでやってくれます。

---

## このダッシュボードでの実例

```
「高崎の天気予報を週間表示に変えて」
→ dashboard.html のAPIコールと表示部分を修正してGitにpush
```

```
「日経平均の色が見づらいのでもっとはっきりした色に変えて」
→ CSSを探して変更
```

```
「スマホで見たとき文字が小さすぎる。フォントサイズを大きくして」
→ モバイル用スタイルを調整
```

---

## スマホで使うときのコツ

- **音声入力を活用する** - キーボード入力より早い
- **日本語でOK** - 難しい英語の専門用語は不要
- **「〇〇して」で伝わる** - 短い指示でも動いてくれる
- **確認しながら進める** - 「まず現状を見せて」→「じゃあ変えて」と段階的でもOK

---

## 注意点

- 大きな変更（複数ファイルにまたがる改修など）は画面が小さいと確認しづらい
- コードのレビューはPCの方がやりやすい場合もある
- ただし「指示を出す」だけならスマホで十分
